# Tiny Swords RPG 🗡️

A charming 2D action-adventure RPG with precise combat and deep exploration mechanics, built in Unity.

## 🕹️ Controls

| Action          | Keyboard |
|-----------------|----------|
| Movement        | WASD     |
| Attack          | K        |
| Use Item        | L-Click  |
| Drop Item       | R-Click  |
| Stats           | Esc      |
